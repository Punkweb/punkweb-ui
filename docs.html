PunkwebUI

v0.6.0 — Lightweight CSS framework with classless defaults, utility classes, and essential components.

Getting Started

PunkwebUI is a lightweight (~25kB gzipped) CSS framework available as a standalone CSS file or as an npm package for SCSS-based build systems.

Standalone CSS

Download the latest release and link the minified CSS file:

<link rel="stylesheet" href="/path/to/punkweb-ui.min.css" />

npm / SCSS

npm install punkweb-ui

Then import the SCSS entry point in your project:

@import 'punkweb-ui/scss/punkweb-ui';

Dark Mode

Add data-theme="dark" to your <html> or <body> tag.

<html lang="en" data-theme="dark">

PunkwebUI is themed entirely via CSS custom properties. Override any of the variables below in your own stylesheet to customise the look of the framework without touching source files.

Dialog

Modal/dialog layout component. Includes a header, scrollable body, and actions footer. Positioning and backdrop are left to you (the CSS handles internal layout only).

Confirm Delete

Are you sure you want to delete this item? This action cannot be undone.

<div class="pw-dialog">
  <div class="pw-dialog-header">Confirm Delete</div>
  <div class="pw-dialog-body">
    <p>Are you sure?</p>
    <div class="pw-dialog-actions">
      <button>Cancel</button>
      <button class="pw-button primary">Delete</button>
    </div>
  </div>
</div>

Horizontal
<form class="pw-form horizontal">…</form>
Multi-column with pw-form-row
<form class="pw-form">

    <div class="flex-1">
      <div class="pw-form-group">

Responsive Utilities

All utility classes support responsive breakpoint prefixes. Prefix any utility with a breakpoint name to apply it only at that screen size and above.

Prefix Breakpoint Example
xs: ≥ 512px (32rem) .xs:flex
sm: ≥ 640px (40rem) .sm:grid-cols-2
md: ≥ 768px (48rem) .md:hidden
lg: ≥ 1024px (64rem) .lg:text-xl
xl: ≥ 1280px (80rem) .xl:p-8
